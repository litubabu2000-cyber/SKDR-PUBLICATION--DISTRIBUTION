# **App Name**: LearnScape

## Core Features:

- Book Catalog: Display a catalog of educational books, searchable by title, author, and subject.
- Past Exam Papers: Provide access to previous years' question papers (PYQ) for various exams.
- Video Lectures: Embed and organize video lectures by chapter and topic, possibly sourced from YouTube or other platforms.
- Chapter-wise Mock Tests: Offer mock tests for each chapter to assess understanding.
- Topic-wise MCQ Practice: Provide multiple-choice questions (MCQ) for practicing specific topics.
- Weekly Tests: Administer weekly all-India and state-wise tests, with a leaderboard, to gauge student performance.
- Exam Updates: Deliver the user with the most recent information regarding the all the examinations.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to represent intellect, knowledge, and sophistication.
- Background color: Light Lavender (#E6E6FA), a desaturated tint of indigo, to provide a calming, distraction-free backdrop.
- Accent color: Gold (#FFD700) for highlighting important elements like exam names or correct answers.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text; Poppins will give a clean modern look while PT Sans adds slight warmth for readability.
- Use clear, modern icons to represent different sections, exams, and content types.
- Implement a clean and intuitive layout for easy navigation.
- Subtle animations when transitioning between pages.
